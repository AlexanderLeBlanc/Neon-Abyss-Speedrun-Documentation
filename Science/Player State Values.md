# Player State Formulas

| Name | Equation |
| :-: | :- |
| `FinalMoveSpeed` | `(MoveSpeed + TempMoveSpeed + LayerMoveSpeed) * MoveSpeedDecay` |
| `HurtInvincibilityTime` | `1` |
| `FinalBulletRPM` | `TempBulletRPM + BaseRPM + ExtraBulletRPM + LauncherRPM + LayerBulletRPM` |
| `FinalBulletLifeSpanMult` | `BulletLifeSpanMult + TempBulletLifeSpanMult + ExtraBulletLifeSpan + LayerBulletLifeSpanMult` |
| `FinalBulletDamageMult` | `TempBulletDamageMult + BulletDamageMult + ExtraBulletDamageMult + LayerBulletDamageMult` |
| `FinalBulletSizeMult` | `1 + (TempBulletDamageMult + BulletDamageMult + ExtraBulletDamageMult - 1)` |
| `FinalBulletSpeedMult` | `BulletSpeedMult + TempBulletSpeedMult + ExtraBulletSpeedMult` |
| `FinalBulletCount` | `TempBulletCount + BulletCount + ExtraBulletCount` |
| `FinalMeleeDamage` | `MeleeBaseDamage * (MeleeDamageMult + ExtraMeleeDamageMult + TempMeleeDamageMult)` |
| `FinalMultipleJumpCountLimit` | `ExtraMultipleJumpCountLimit + multipleJumpCountLimit` |
| `FinalMeleeBulletLifeSpanMult` | `MeleeBulletLifeSpanMult + ExtraMeleeBulletLifeSpanMult + ActorMeleeBulletLifeSpanMult` |
| `FinalMeleeBulletCount` | `MeleeBulletCount + ExtraMeleeBulletCount + ActorMeleeBulletCount` |
| `FinalMeleeBulletSizeMult` | `1 + (MeleeDamageMult + ExtraMeleeDamageMult + TempMeleeDamageMult - 1) * 0.5` |
| `FinalJumpForce` | `TempJumpForce + JumpForce` |
| `FinalKanonMeleeDamage` | `MeleeBaseDamage * (MeleeDamageMult + ExtraMeleeDamageMult + TempMeleeDamageMult)` |
| `HasMeleeLauncher` | `MeleeBulletLifeSpanMult + ExtraMeleeBulletLifeSpanMult + MeleeBulletCount + ExtraMeleeBulletCount + MeleeBulletSizeMult + ExtraMeleeBulletSizeMult` |
| `Change me` | `1 + 2 * 3 / (4 - 5)` |
| `FinalMeleeDamageMult` | `MeleeDamageMult + ExtraMeleeDamageMult` |
| `FinalJumpLevel` | `JumpLevel + JumpLevelForPowerUp + LayerJumpLevel` |
| `FinalMeleeCheckDistance` | `MeleeCheckDistance + ExtraMeleeCheckDistance` |
| `SpriteDamage` | `100 * (1 + SpriteSizeLevel * 0.3)` |
| `FinalSpriteFlySpeedMult` | `SpriteFlySpeedMult` |
